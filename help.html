热门常见问题

如何定位附近休息站点？

打开地图视图，休息站点会以绿色图标显示。点击"休息站"筛选按钮可只显示休息站点。您也可以使用搜索功能查找特定休息站。

点击站点图标可查看详细信息，如设施、开放时间等。系统会根据您的位置自动推荐最近站点。

意外伤害保险如何申请理赔？

1. 在事故发生后24小时内通过骑手中心提交事故报告
2. 准备医疗证明、事故照片和费用单据
3. 在"保险理赔"板块填写申请表并上传材料
4. 保险公司将在3-5个工作日内审核处理

如何申请健康体检福利？

当您当月累计完成200单以上后：
1. 进入骑手中心 -> 福利保障板块
2. 选择"健康体检"服务
3. 选择合作医院和预约时间
4. 系统会生成体检二维码，凭码即可体检

遇到导航错误怎么办？

1. 确认GPS定位是否准确
2. 在"地图设置"中清除缓存并刷新地图
3. 如果路线错误，使用"路径规划"功能手动调整
4. 点击问题反馈按钮报告具体位置和问题描述
5. 联系技术支持获取实时帮助

薪资结算规则是怎样的？

薪资每周结算一次，每周一发放上周收入
收入构成：基础配送费 + 时段补贴 + 距离补贴 + 好评奖励
可在骑手中心查看每日明细和累计收入
提现到银行卡需1-3个工作日，无手续费

如何查看历史配送数据？

1. 进入骑手中心 > 数据统计
2. 选择时间范围（日/周/月）
3. 可查看完成单量、里程、收入等数据
4. 可导出数据为Excel文件用于个人记录
